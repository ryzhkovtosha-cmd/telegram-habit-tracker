# Чат-бот для трекинга привычек

Телеграм-бот, который помогает пользователям вырабатывать и отслеживать полезные привычки. Сервис создан по заказу «Центра по подбору психологов» как инструмент для клиентов на пути самосовершенствования.

## Возможности

- Регистрация через Telegram (без паролей, по ID пользователя).
- Создание, редактирование и удаление привычек.
- Ежедневный список привычек с возможностью отметить «выполнено» или «не выполнено».
- Автоматический перенос активных привычек на следующий день (до 21 выполнения).
- Напоминания о необходимости отметить привычки (каждый день в 9:00).
- Безопасная аутентификация через JWT-токены.
- Полное API для внешних интеграций (доступно через Swagger).

## Архитектура

Проект состоит из трёх компонентов, запускаемых в Docker-контейнерах:

1. **PostgreSQL** — база данных.
2. **FastAPI (backend)** — REST API, обрабатывает запросы от бота и предоставляет документацию.
3. **Telegram-бот** — интерфейс пользователя, общается с API.

Все части связаны через внутреннюю сеть Docker. Управление миграциями БД осуществляется Alembic.

## Запуск проекта

### 1. Клонирование репозитория

git clone https://github.com/ryzhkovtosha-cmd/telegram-habit-tracker.git
cd telegram-habit-tracker

### 2. Настройка переменных окружения

Создайте файл .env в корне проекта (рядом с docker-compose.yml). Добавьте в него токен вашего Telegram-бота:
ini

BOT_TOKEN=123456:ABC-DEF1234ghijklmnopQRSTUVwxyz

    Файл .env уже добавлен в .gitignore и не попадёт в репозиторий.

### 3. Запуск через Docker Compose

docker-compose up -d

После первого запуска API будет доступен на порту 8001 (http://localhost:8001).
Swagger-документация откроется по адресу http://localhost:8001/docs.

### 4. Остановка

docker-compose down

## Использование Telegram-бота

Найдите своего бота в Telegram и отправьте команду /start. После авторизации вам станут доступны следующие команды:
Команда	Действие
/start	Регистрация / авторизация
/habits	Посмотреть список активных привычек
/add	Добавить новую привычку
/edit	Изменить название или описание привычки
/delete	Удалить привычку
/daily	Список привычек на сегодня (с кнопками)
/help	Показать справку
